h1. Tarea 1: Entorno de desarrollo

h2. Realiza un fork del repositorio de GitHub: https://github.com/josedom24/django_tutorial.

Una vez realizado el fork vamos a clonar por ssh el repositorio:
<pre>git clone https://github.com/josedom24/django_tutorial</pre>

h2. Crea un entorno virtual de python3 e instala las dependencias necesarias para que funcione el proyecto.
Creamos el entorno:
<pre>$ virtualenv entorno -p python3</pre>
Instalamos las dependencias:
<pre>(entorno)$ pip install -r requirements.txt</pre>

h2. Comprueba que vamos a trabajar con una base de datos sqlite. ¿Qué fichero tienes que consultar?.
¿Cómo se llama la base de datos que vamos a crear?
Tenemos que consultar el fichero settings.py que se ubica en el directorio django_tutorial.
La base de datos que vamos a crear se llama db.sqlite3.

h2. Crea la base de datos. A partir del modelo de datos se crean las tablas de la base de datos.
Ejecutamos el siguiente comando para generar la base de datos con sus respectivas tablas:
<pre>(entorno)$ python3 manage.py migrate</pre>
Debemos instalar sqlite3 para poder conectarnos a la base de datos.
<pre>(entorno)$ sudo apt install sqlite3 -y
Comprobamos las tablas con:
<pre>(entorno)$ sqlite3 db.sqlite3
Salida del comando:
!tablas.png!

h2. Crea un usuario administrador.
El usuario se crea con el siguiente comando:
<pre>(entorno)$ python3 manage.py createsuperuser</pre>
Comprobacion:
<pre>(entorno)$ select * from auth_user;
Salida del comando:
!creacionadministrador.png!
